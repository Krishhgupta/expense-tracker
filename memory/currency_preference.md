**Currency:** Use INR (₹) for all monetary values in the expense tracker.

**Why:** This is an Indian expense tracking application, so all amounts should be displayed and stored in Indian Rupees.

**How to apply:** When implementing or modifying any code that handles money (expense amounts, totals, currency symbols), use INR/₹ instead of USD/$.
